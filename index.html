LIST OF GIT COMMANDS

  1. git init-initializes a git repository in local directory
  2. git add (filename)-add file to staging
  3. git status-This command will show the status of the current repository including staged, unstaged, and untracked files.
  4. git commit -m "your commit message here"-Creates a new commit with the changes in the staging area and specifies the commit message inline.
  5. git commit -a -m"your commit message here"-You can add and commit tracked files with a single command by using the -a and -m options.
